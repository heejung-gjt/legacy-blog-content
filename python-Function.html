1.1 Python 함수

1-1 함수

1.함수는 반복되는 작업을 수행할때 반복적으로 사용되는 부분을 함수로 정의하여 사용하는 기능이다. 함수의 네이밍은 소문자로 단어 사이에 밑줄 _ 을 사용하여 연결한다

2.def는 함수를 만들 때 사용하는 예약어 이다

def add(a, b):
    return a + b

num1 = int(input())
num2 = int(input())
result = add(num1, num2)
print(result)


위의 함수이름은 add 이고 a,b 2개의 값을 받으면 2개의 값을 더한 값이 리턴되는 함수이다 num1과 num2의 정수값을 입력받고 해당 값들을 더한 결과를 result 변수에 저장 하여 출력한다

1-2 전달값과 반환값

1.함수에 입력으로 전달하는 값을 parameter(인자)라고 한다. 전달되는 parameter를 괄호 안에 전달하여 함수를 호출한다. 함수의 종료를 명시할때 반환값을 주고 싶을 경우에는 return으로 값을 반환해준다

def money(input, output):  #input , output  값 전달
    
    return input - output  #반환값 

moneyIn = int(input('입금금액 :'))
output = int(input('출금금액 :'))

result = money(moneyIn, output)
print('입출금액은 각각 {0}원 {1}원 입니다, 남은돈은 {2} 원 입니다'\
    .format(moneyIn,output, result))

#답 :
#    입금금액 :100000
#    출금금액 :4500
#    입출금액은 각각 100000원 4500원 입니다, 남은돈은 95500 원 입니다


1-3 기본값과 키워드값

1.기본값 : 함수에 전달되는 parameter 중 디폴트로 지정된 값을 사용할 수 있다. parameter를 명시하지 않을 경우 기본값으로 대체한다

2.키워드값 : parameter에 값을 전달할때, 파라미터의 이름을 명시하여 전달 할 수 있다. 순서를 다르게 전달해도 순서대로 인식이 가능하다

#기본값
def add(x=10,y=20,z=30):
    result = x + y + z
    return result

print('x, y, z 명시 :',add(1,1,1))
print('x, y 명시 :',add(1,1))
print('x 명시 :',add(1))
print('명시 x :',add())

#키워드값
def add(x,y,z):
    result = x - y + z
    return result

print(add(x=100, z=30, y=200))  #답 : -70
print(add(100, 30, 200))  #답 : 270


1-4 가변인자

1.parameter의 입력 개수를 알 수 없거나 0~n개의 parameter를 받고 싶을때 사용한다
parameter명 앞에 *을 붙여서 가변길이임을 표시한다

def add(*num):
    result = 0
    for i in num:
        result +=i
    return result

result1 = add(1,2)  
result2 = add(7,8,9,10)  

print(result1) # 답 : 3
print(result2) # 답 : 34


1-6 지역변수와 전역변수

1.지역변수 : 함수의 안에서 생성이 되는 변수이다. 외부에서는 참조할 수 없는 변수이다

def nums():
    num1 = 10
    num2 = 20
    print(num1+num2)  #가능 (함수 nums안에 생성이 되어 있기 때문에 가능하다)

print(num1+num2)  # 불가능  (함수 nums 외부에 생성이 되어 있기 때문에 불가능한다)


2.전역변수 : 함수 밖의 전역 공간에서 선언되는 변수이다. 함수 안에서 참조가 가능하다

num1 = 100
num2 = 10
def nums():
    print(num1+num2)  #가능 (변수값들이 전역에 생성되어 있기 때문에 가능하다)

print(num1+num2)  #가능 (변수값들이 전역에 생성되어 있기 때문에 가능하다)
