리눅스 기본 명령어

👉 Linux 명령어 정리


디렉토리 목록 확인

  • ls
  • ls -al : 숨겨진 파일까지 확인할 수 있다

디렉토리 생성

  • mkdir <디렉토리명>
  • mkdir -p <디렉토리명/…> : 여러 디렉토리를 생성할 수 있다

디렉토리 이동

  • cd <디렉토리명>
  • cd .. : 부모 디렉토리로 이동한다

디렉토리 삭제

  • rm -r <디렉토리명>
  • rmdir <디렉토리명> : 비어있는 디렉토리를 삭제한다

파일 생성

  • touch <파일명>

파일 이동

  • mv <파일이름> <이동할 폴더나="" 파일이름="">

파일 이름변경

  • mv <파일이름> <바꿀 파일이름="">

파일 삭제

  • rm <파일이름>

파일 내용보기

  • cat <파일이름>

명령창 내용 삭제

  • clear
