Ethiopian Multiplication

  • example input : 14 5
  • exmaple output : 70

풀어야 할 목표

14 5 -> 70
7 10 keep
3 20 keep
1 40 keep
(10 + 20 +50) -> 70

코드구현

num1, num2 = map(int,input().split())
result = 0

while num1 > 0:
    if num1 % 2 == 0:  #num1의 짝수를 찾는다
    print(f'pass {num1} {num2}')
    else:
        result += num2 # 짝수일 경우 result에 누적으로 더해준다 
        print(f'keep {num1} {num2}')    
    num1 //= 2 
    num2 *= 2
    
print(result)  #홀수일때 곱해졌던 num2 합계


Monty Hall Simulation

몬티홀 정리

1. 문3개를 설정한다 doors = [0,0,1]. 이때 1을 차라고 설정을 하겠다

2. 3개의 문에 1의 위치(차 위치)를 랜덤으로 배치하기 위해서 shuffle 함수를 이용해준다. random.shuffle(doors)

3. 사용자가 3개의 문 중 한개를 랜덤으로 뽑는다. 이때 랜덤으로 뽑는 2가지 방법이 존재한다.
(1) 이미 문을 랜덤하게 섞기 때문에 (2번에서) pop을 이용하여 맨 마지막 인덱스를 뽑아준다
(2) 문을 랜덤하게 뽑을 수 있는 random.choice(doors) 를 이용하여 뽑아준다

4. 사회자가 문안에 있는 0을 하나 뽑아준다.이때 remove는 지정한 숫자가 리스트에 여러개 있는 경우에 가장 앞에 있는 중복되는 숫자 하나만 삭제한다. 즉, 0이 여러개여도 한개만 삭제된다는 것을 알 수 있다.

5. 이후 사용자가 문을 바꾸었을때 바꾸지 않았을때를 조건문으로 처리해준다
if문을 사용하여 사용자가 처음 선택했던 숫자가 1일 경우에는(차) 바꾸지 않는 것이 이득이므로 바꾸지 않았을때의 변수 stay에 +1를 누적해준다
만약 처음 선택했던 숫자가 1이 아닌 경우(염소)에는 바꾸는 경우가 더 이득이기 때문에 바꿨을때의변수 switch에 +1를 누적해준다

6. 그후 %를 구해준다. 만약 10000번 반복하였다면 바꾸지 않았을때의 확률 (stay/trial)이며 바꾸었을때의 확률 (switch/trial)이다

코드 구현

import random

stay = 0
switch = 0
trial = 10000

for _ in range(trial):
    doors = [0,0,1]
    random.shuffle(doors)

    users_choice = doors.pop()

    doors.remove(0)

    if users_choice == 1:
        stay += 1
    else:
        switch += 1
print(f'stay : {stay/trial}, switch{switch/trial}')
