👉 Operators


[Task]

식사가격, 팁비율(팁으로 추가되는 식사가격의 비율), 세금비율(세금으로 추가되는 식사가격의 비율) 을 계산해 총 식사의 비용을 구하는 내용이다

[Example]

팁은 15% * 100 = 15, 세금은 8% * 100이다.

[Function Description]

solve에는 다음 매개 변수가 있다
int meal_cost: 팁 및 세금 전 음식 비용
int tip_percent : 팁비율
INT tax_percent : 세금비율
가장 가까운 정수로 반올림하여 계산된 값을 출력해라

[Input Format]

첫번째 줄에는 세금 및 팁 전 식사 비용
두번째 줄에는 백분율 mealCost팁으로 추가
세번째 줄에는 백분율 mealCost 세금으로 추가


[Sample Input]

12.00
20
8


[Sample Output]

15


[Explanation]


[Code]

import math
import os
import random
import re
import sys

# Complete the solve function below.
def solve(meal_cost, tip_percent, tax_percent):
    tip = meal_cost * (tip_percent / 100)  #팁으로 추가되는 식사가격 비율 
    tax = meal_cost * (tax_percent / 100)  #세금으로 추가되는 식사가격 비율
    total_cost = round(meal_cost + tip + tax)
    print(total_cost)

if __name__ == '__main__':
    meal_cost = float(input())
    tip_percent = int(input())
    tax_percent = int(input())
    solve(meal_cost, tip_percent, tax_percent)
