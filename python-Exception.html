Python 예외처리

1-1 예외처리란

1.예외처리란 구문에 오류가 발생하였을때 비정상적으로 종료되는 것을 방지하고 정상적인 오류 발생으로 처리해주는 기능이다

오류가 나는 대표적인 예 01 - 0으로 다른 숫자를 나누는 경우
ZeroDivisionError이라는 오류가 발생한다

num1 = 4
num2 = 0
result = num1 / num2
print(result)

'''
발생오류
Traceback (most recent call last):
  File "c:\....py", line 3, in <module>
    result = num1 / num2
ZeroDivisionError: division by zero
'''


오류가 나는 대표적인 예 02 - 존재하지 않는 리스트의 값을 얻으려고 하는 경우
IndexError라는 오류가 발생한다

list = [1,2,3]

print(list[3])

'''
발생오류
Traceback (most recent call last):
  File "c:\......py", line 3, in <module>
    print(list[3])
IndexError: list index out of range
'''


try ~ except 문
위에서 나는 오류뿐만 아니라 그 외의 오류가 발생했을시 정상적으로 오류를 처리해준다

num1 = 4
num2 = 0
list = [1,2,3]

try:
    result = num1 / num2
    print(result)
    print(list[3])
except:         #오류 발생시 except 내용 출력한다
    print('오류가 발생하였습니다')
'''
결과
오류가 발생하였습니다

'''


try ~ 발생 오류 except 문
오류를 처리해 줄 때 발생 오류만 포함한 except문을 만들 수 있다

list = [1,2,3]

try:
    print(list[3])
    
except IndexError:
    print('인덱스 오류입니다')

'''
결과
인덱스 오류입니다

'''


try ~except ~ finally
오류를 처리해 줄 때 try ~ except ~ finally 를 이용하여 처리해준다. 이때 finally절은 try문 수행 도중 오류 여부 상관없이 항상 수행된다

list = [1,2,3]

try:
    print(list[3])
    
except IndexError:
    print('인덱스 오류입니다')

finally:
    print('연산이 끝났습니다')

'''
결과
인덱스 오류입니다
연산이 끝났습니다

'''


2-1 raise

  1. 오류를 일부러 발생시켜야 할 경우가 있을때 raise명령어를 사용하여 강제로 발생시킨다
try:
    print('한자리 숫자 나누기 계산기')
    num1 = int(input('첫번째 숫자'))
    num2 = int(input('두번째 숫자'))

    if num1 >= 10 or num2 >= 10 :
        raise ValueError  #ValueError 라는 오류 발생시킨다 
    
    print(f'{num1} / {num2} = {num1/num2}')

except ValueError:
    print('값을 잘못 입력 ! 한 자리 숫자만 입력')
 
 '''
 발생오류
한자리 숫자 나누기 계산기
첫번째 숫자10
두번째 숫자2
값을 잘못 입력 ! 한 자리 숫자만 입력
 '''


3-1 사용자 정의 예외처리

1.오류 발생 시 사용자가 직접 에러를 정의하여 오류를 처리 할 수 있다

class BigNumberError(Exception):   #사용자가 직접 BigNumberError이라는 오류를 생성한다
    def __init__(self,msg):
        self.msg=msg

    def __str__(self):
        return self.msg

try:
    print('한자리 숫자 나누기 계산기')
    num1 = int(input('첫번째 숫자'))
    num2 = int(input('두번째 숫자'))

    if num1 >= 10 or num2 >= 10 :
        raise BigNumberError(f'입력값 : {num1} , {num2}')  #해당 오류에 메시지를 전달한다
    
    print(f'{num1} / {num2} = {num1/num2}')

except BigNumberError as err:
    print('ERROR ! 한 자리 숫자만 입력') 
    print(err)   #BigNumberError를 출력하면 해당 예외처리에 들어가 있는 메시지가 출력된다
 '''
 발생오류
한자리 숫자 나누기 계산기
첫번째 숫자10
두번째 숫자2
ERROR ! 한 자리 숫자만 입력
입력값 : 10 , 2
 '''
