(프로그래머스 with python) 문자열 압축


Level2 - 문자열 압축


https://programmers.co.kr/learn/courses/30/lessons/60057


문제를 처음 봤을때 쉽게 아이디어가 떠오르지 않았다.. 결국에는 다른 사람의 코드로 도움을 받게되었는데 빨리 내 머리와 손으로 직접 아이디어를 떠올리고 문제를 푸는 희열을 맛보고 싶다..

이 문제의 첫번째 풀이방법은 카운트 를 하는것이다. 문자열이 주어지면 현재의 단어와 이전의 단어가 같은지 다른지를 비교해나가는며 카운트를 하는것이 핵심이다. 이때 문자열을 가장 짧게 압축한 값이 반환되기 때문에 하나로 압축했을때부터 시작해서 일일히 압축한 값을 확인해야 한다

  1. 우선 문자열로 되어있는 값들을 리스트에 하나씩 넣어준다
  2. 새롭게 만든 리스트를 가지고 for문을 돌린다. 이때 압축시키는 개수가 1 ~ len(n) 까지 늘어나기 때문에 이를 고려해서 for문을 돌려주어야 한다
  3. 압축시키는 개수만큼 리스트에 값을 넣어주는 for문을 통해 splited리스트에 값을 담아준다 ex) 2개씩 압축 aabb -> splited = [‘aa’, ‘bb’]
  4. aabbcc이고 하나씩 잘라 비교한다면 a와 a를 비교 후 값이 같으면 cnt + 1, a와 b를 비교 후 값이 다르면 이전값과 cnt한 값을 새로운 문자열에 추가해준다. (a vs a , cnt + 1 , a vs b , new_str += cnt + prev)
  5. 마지막 값이 돌때는 for문을 벗어난 곳에서 splite[-1]로 4번과 같은 처리를 해주어야 한다
def solution(s):
    answer = ''
    s_list = [i for i in s] - 1
    new_str = ''
    len_list = []
    
    if len(s) == 1:
        return 1
    
    for i in range(1, len(s_list)): - 2
        cnt = 1
        compressed = ''
        splited = []
    
        for j in range(0, len(s_list), i): - 3
            splited.append(s[j: j + i])
        
        for k in range(1, len(splited)):
            p, c = splited[k - 1], splited[k] - 4
            if p == c:
                cnt += 1
            else:
                compressed += str(cnt) + p if cnt > 1 else p
                cnt = 1
        
        compressed += str(cnt) + splited[-1] if cnt > 1 else splited[-1] - 5
        len_list.append(len(compressed))
    
    return min(len_list)

solution("aabbaccc")
