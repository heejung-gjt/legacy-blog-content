BlackJack 게임 Procedural Programming 구현

👉 게임규칙

  1. 블랙잭 코드는 Procedural Pradigm으로 작성되어야 한다.

  2. 카드 덱 구성은 2~11 까지 정수형 숫자로 표현되며, 문양은 생략한다.

  3. 덱은 2~11까지의 숫자에 대한 카드가 4장씩 총 40장으로 이루어져 있다.

  4. 11 카드는 Ace이며, 1또는 11의 숫자값을 가진다. 11로 시작하여 사용자나 딜러의 숫자 카드의 합이 21을 초과할 경우, 1로 치환될 수 있다.

  5. 게임참여자는 뱅크(딜러)와 사용자(유저)로 이루어져 있다.

  6. 프로그램이 시작되면 카드 셔플을 3회 진행하며, 유저가 원할 경우 한 번 더 셔플할 수 있다.

  7. 카드 셔플이 완료된 후에는 그 순서가 고정되며, 덱에 남아있는 카드가 10장 아래일 경우 덱을 초기화 하여 다시 셔플절차를 수행한다.

  8. 게임이 시작되면 뱅크와 사용자가 1장씩 카드를 받게 되며, 이 카드는 오픈되어 전달된다.(출력하여야 한다.)

  9. 사용자가 먼저 게임을 치르며, 게임은 숫자의 조합이 21과 일치하거나 21을 넘지않는 가장 가까운 사람이 승리한다.

  10. 사용자는 게임이 시작되면 덱에서 숫자를 하나 전달받는다. 오픈카드 부터 받은 순서대로 표현하며, 정렬은 하지 않는다.

  11. 사용자에게 카드를 더 받을지 멈출지를 물어보아야하며, 만약 받은 카드의 숫자 합이 21을 초과할 경우 사용자의 턴은 종료되며, 딜러가 21을 초과하지 않는 한 이길 수 없다.

  12. 사용자가 처음 받은 카드가 10, 11 일 경우, 블랙잭으로 딜러의 결과와 관계없이 게임을 승리한다.

  13. 사용자의 턴이 끝난 후 딜러의 턴이 시작되며, 숫자의 합이 16을 넘을 때까지 카드를 받아야 한다.

  14. 사용자가 21을 넘긴 후 딜러도 21을 넘긴 경우, 게임은 비긴 것으로 처리한다.

  15. 사용자와 딜러 모두 21을 넘기지 않은 경우, 21에 가장 가까운 객체가 승리한다.

  16. 이 프로그램은 CLI로 동작해야 한다.

👉 PreudoCode

  1. 카드 덱 구성은 2-11 범위의 총 40장으로 이루어져 있다. for문을 이용해 정수 2-11을 각각 4개씩 배열(card_deck)에 넣는다.

  2. 랜덤하지 않은 배열(card_deck) 랜덤하게 3번 섞는다. for문으로 범위를 3으로 정해 3번동안 shuffle()을 이용해 섞어준다.

  3. 사용자가 한번의 셔플을 정할 수 있으므로 y/n으로 입력값을 받아 y인 경우 shuffle()이용해 한번 더 섞어준다.

  4. 카드셔플 완료후 딜러와 사용자 각각 카드덱에서 랜덤한 숫자값 하나씩 받아 오픈한다. 딜러(delar_card)와 사용자(user_card)의 배열을 새로 생성해 카드덱에서 랜덤하게 뽑은 값은 append 해주고 card_deck 배열에는 해당 값을 remove 해준다.

  5. 사용자의 turn이 먼저이며, 덱에서 랜덤한 숫자를 하나 전달받는다. 전달받은 값 사용자의 배열인(user_card)에 차례대로 append해준다.
    이때 사용자의 값에서 첫번째 숫자가 11인 경우 , 2개의 합이 21을 넘는경우 11을 ->1 로 변경한다. if문을 이용해 user_card[0]에 11인지 아닌지를 먼저 판단한 후 맞을경우 user_card[0]=1로 바꾸어준다.

  6. 사용자가 카드를 더 받을지(hit) 멈출지(stay)를 물어본 후 새롭게 받는 경우 user_card의 값이 21을 넘지않는지 판단 후 넘지않는 경우 사용자가 랜덤한 카드덱을 뽑아 자신의 배열(user_card)에 넣는다. 이때 user_card의 합이 21 초과인 경우 딜러의 turn으로 넘어가게 되며, 딜러의 값에 크기에 따라 무승부,승자, 패자가 결정된다.

  7. 딜러의 턴으로 넘어온 후 딜러는 카드의 합계가 16을 넘을때까지 카드를 받아야한다. while문으로 dealar_card의 합계가 16을 넘지 않는동안 반복해서 card_deck 의 랜덤한 숫자값을 하나씩 뽑아온다.

  8. 딜러의 카드 합계 16을 넘게되면 유저의 카드값과 비교한다. 이미 앞에서 유저의 카드값이 21을 초과하였기때문에 딜러의 카드합계가 21을 초과해야지만 무승부가 된다. 만약 21을 초과하지 않으면 딜러가 우승자가 된다.

  9. 만약 유저가 멈추는 것을 선택한다면 딜러의 턴으로 바로 넘어가게 된다. 딜러는 7번을 반복하고 딜러의 카드합이 16을 넘기게 되면 유저의 카드합계와 딜러의 카드합계를 비교하여 21에 가장 가까운 사람이 승리하게 된다.

  10. card_deck이 감소할때마다 카드덱 수를 카운트하여 10보다 작아지는경우 셔플절차를 수행한다. card_deck를 len()을 이용하여 길이를 카운트하여 해당 길이가 10보다 작은경우 1~2번을 수행한다.

👉 코드로 구현하기

#pop() 이용해서 card_deck 뽑는 방식
import random


def init():  # 40개의 3번 셔플한 랜덤한 카드덱 생성하기
    card_deck = [i for i in range(2,11+1) for _ in range(4)]
    for _ in range(3):
        random.shuffle(card_deck)
    print('*'*25,'BLACK JACK GAME START','*'*25,
          '\n\nCard Deck Shuffling..\n')
    return card_deck 


def user_choose_shuffle(card_deck): # 사용자가 원할 경우 한번더 셔플
        card_shuffle = input('Complete card deck shuffle ! [one more? : y/n]:')
        if card_shuffle =='y':
            print('\n--- Card Deck Shuffle ---')
            random.shuffle(card_deck)
        return card_deck    
            
        
def game_start(): #게임 시작. 딜러와 사용자 오픈하는 카드 1장씩 부여
    user_card = []
    dealar_card = []
    user_card.append(card_deck.pop())
    dealar_card.append(card_deck.pop())
    print('\n','*'*30,'FIRST TURN','*'*30,'\n')    
    print(f'USER Card : {user_card}\n')
    print(f'DEALAR Card : {dealar_card}\n')
    return user_card, dealar_card


def first_user_turn(user_card,card_deck): # 사용자 먼저 게임 시작, 덱에서 랜덤한 숫자 하나 전달
    user_card.append(card_deck.pop())
    if user_card[0] == 11:  # 처음값 11, 합계 20초과시 11->1로 변경
        if sum(user_card) > 20:
            user_card[0]=1
            
    if sum(user_card) == 21:  
        print('🌟BLACKJACK🌟 USER WIN !')
        return
    
    print('\n','*'*28,'FIRST USER TURN','*'*28,'\n')
    print('Receive a one Card..\n')
    print(f'USER Card : {user_card}\n')
    return user_card, card_deck


def user_turn(user_card,card_deck): # 사용자 turn
    user_card.append(card_deck.pop())
    card_deck = remain_card(card_deck)
    print('*'*32,'USER TURN','*'*32,'\n')
    print('Receive a one Card..\n')
    print(f'User_Card : {user_card}')
    return user_card,card_deck 

    
def dealar_turn(dealar_card, card_deck): # 딜러 turn
    if dealar_card[0] == 11:  # 처음값 11, 합계 20초과시 11->1로 변경
        if sum(dealar_card) > 20:
            dealar_card[0]=1
            
    while sum(dealar_card) < 17 :
        print(f'Dealar Receiving a Card..\n')
        dealar_card.append(card_deck.pop())
        card_deck = remain_card(card_deck)
    print(f'Total Dealar Card : {sum(dealar_card)}\n')
    return dealar_card, card_deck


def ask_user(dealar_card,user_card, card_deck): # ask hit or stay 
    while True:
        print('\n','*'*29,'HIT ? ASK USER','*'*29,'\n')
        answer = input('One more card? [y/n] :')
        if answer == 'n':
            print('\n--- STAY ---')
            print('*'*31,'DEALAR TURN','*'*31,'\n') 
            dealar_card, card_deck = dealar_turn(dealar_card, card_deck)
            print(f'>>>>>TOTAL USER CARD SUM : {sum(user_card)}')
            print(f'>>>>>TOTAL DEALAR CARD SUM {sum(dealar_card)}\n')            
            diff_card_sum(dealar_card, user_card)
            break
            
        elif answer == 'y':
            print('\n--- hit ---')
            user_card, card_deck = user_turn(user_card, card_deck)
            if sum(user_card) > 20:
                print('\n--- Total card exceeds 21 ---\n')
                print('*'*31,'DEALAR TURN','*'*31,'\n') 
                dealar_card, card_deck = dealar_turn(dealar_card, card_deck)
                if sum(dealar_card) > 20:
                    print('DEAD HEAT ❗')  
                else:
                    print('🤣DEALAR WIN🤣')
                break
                

def diff_card_sum(dealar_card, user_card): # 유저와 딜러 카드합계 diff
#     print(f'유저 카드 합계{sum(user_card)}')
#     print(f'딜러 카드 합계{sum(dealar_card)}')
    if sum(dealar_card) > 20:
        print('🌟USER WIN🌟')
    else:
        if sum(dealar_card) > sum(user_card):
            print('🤣DEALAR WIN🤣')
        else:
            print('🌟USER WIN🌟')

            
def remain_card(card_num):  # 카드덱 수 10장 아래일 경우 카드덱 셔플절차 수행
    if len(card_num) < 10 :
        print(f'남은 Card Deck 수 : {card_num}, 10장 이하.. 카드 shuffle 필요')
        card_num = init()
        print(f'카드 shuffle 완료 {len(card_num)}')
    return card_num
