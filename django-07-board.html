class BoardForm(forms.Form): 
    title = forms.CharField(
        error_messages={
            'required':'제목을 입력해주세요'
        },
        max_length=128, label='제목')
    contents = forms.CharField(
        error_messages={
            'required':'내용을 입력해주세요'
        },
        widget=forms.Textarea, label='내용')

  • 📌view에서 로그인한 사용자가 글 쓸 수 있게 설정한다

    1. if request.method == ‘POST’: template으로부터 POST 요청을 받으면 tmeplate에 작성한 데이터를 넣어준다
    2. if form.is_valid() : 받은 데이터가 유효한지 확인한다
    3. board = Board() : 만약 데이터가 유효하다면 model에 있는 Board()의 클래스 변수를 만든다
    4. 만들어진 board의 클래스에 값을 넣어준다
    5. user_id = request.session.get(‘user’) : 이때 사용자는 session에 들어있기 때문에 session을 작성해준다
    6. model Fcuser에 있는 username을 가져온다
board/views.py
from fcuser.models import Fcuser  - ⑥
from django.shortcuts import render, redirect
from .models import Board
from .forms import BoardForm

def board_write(request):
    if request.method == 'POST':  # 제목과 내용이 요청되었을때
        form = BoardForm(request.POST)
        if form.is_valid():
          user_id = request.session.get('user') -⑥
          fcuser = Fcuser.objects.get(pk=user_id) -⑥
          board = Board()
          board.title = form.cleaned_data['title']    - ④
          board.contents = form.cleaned_data['contents']  - ④
          board.writer = fcuser   - ④ , ⑥
          board.save()  # DB에 저장
          
          return redirect('/board/list')  # 전달된 뒤 list 페이지로 전환
    else:
        form = BoardForm()
    return render(request, 'board_write.html',{'form':form})

  • 📌게시글을 보기 위해서는 몇번째 글인지 알아야한다

    1. pk를 입력받는데 해당 pk는 주소로부터 입력을 받아야 한다, 즉 주소로부터 값을 받아 활용한다
    2. path(‘detail//', views.board_detail) : url에서 연결을 할때 정수형 변수 pk를 뒤에 받으면서 views.py에 있는 board_detail을 사용하게 된다
    3. board = Board.objects.get(pk=pk) : pk에 해당하는 글을 가져온다, 즉 입력받은 board의 값을 가져 올 수 있다. 그 뒤 template board에 다시 전달해준다
board/views.py
def board_detail(request, pk):
    board = Board.objects.get(pk=pk)
    return render(request, 'board_detail.html',{'board':board})
