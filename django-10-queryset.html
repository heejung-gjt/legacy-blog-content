Django - DB 조회 queryset

queryset

queryset은 DB에 저장되어 있는 data를 불러오는 방법이다. DB에 정보를 요청하면 작성된 정보가 sql로 매핑되어 queryset이라는 자료 형태로 넘어오게 된다

이를 이용해 DB의 값을 묶어 합을 내거나 숫자를 세는 등의 작업을 할 수 있다
이때 DB는 저장되는 data의 식별을 위해서 자동으로 id를 부여한다 (id=1,2,3…)


queryset 생성

DB를 생성할 model을 작성하고 해당 model에 값을 넣어준다. EX) Board 모델이 생성되어져 있으며 저장된 값이 있는지 확인해보자

board/models.py의 값들을 출력하면 총 7개의 title이 출력된 것을 볼 수 있다. 해당 글들은 django admin을 통해서 생성한 값들이다

장고 쉘 이용하기

$ python manage.py shell

(InteractiveConsole)

board/models.py
class Board(models.Model):
    title = models.CharField(max_length=32,
                                verbose_name='Title')
    contents=models.TextField(verbose_name='Content')
    writer = models.ForeignKey('fcuser.Fcuser',on_delete=models.CASCADE, verbose_name='작성자')  # 모델을 id로 연결하는 필드를 만들어준다 fcuser에 있는 Fcuser와 연결해준다. 사용자가 삭제 한 경우 사용자의 모든 게시글을 삭제해준다
    registered_dttm = models.DateTimeField(auto_now_add=True,
                                verbose_name='등록시간')


다양한 조회 요청 방법

std.objects.all() : std의 queryset 전체를 불러온다

all().order_by(‘id’) : 기본값이며 작성된 순으로 값을 불러온다

all().order_by(‘-id’) : id 역순으로 즉, 최근에 작성한 값부터 불러온다

std.objects.filter() : ()안에 해당하는 조건에 맞는 값을 불러온다

filter(name=”jane”) : 이름이 jane인 값만 불러온다

filter(id=”1”) : id=1 인 값만 불러온다

std.objects.exclude() : ()안에 해당하는 조건을 제외하고 값을 불러온다

exclude(naem=”jane”) : 이름이 jane인 값만 제외하고 불러온다
