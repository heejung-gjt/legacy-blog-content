템플릿 확장이란

장고에서 제공하는 기능 중 하나로 웹사이트 안에 서로 다른 페이지에서 HTML 일부를 재사용 할 수 있는 기능이다
재사용이 가능한 기본 템플릿을 만들어준다. base.html파일로 작성하였다. base 파일에는 meta, link, script등 공통적인 파일들을 상속한다. 해당 템플릿에 가져가서 아래 코드에 적혀있는 block ~ endblock 사이에 원하는 컨텐츠를 작성해주면 된다.

base.html

<html>
    <head>
        <!-- Required meta tags -->
        {% load static %}
        <meta charset="utf-8">
        <meta name="viewport" content="width=device-width, initial-scale=1, shrink-to-fit=no">
        <link rel="stylesheet" href="https://stackpath.bootstrapcdn.com/bootstrap/4.3.1/css/bootstrap.min.css" integrity="sha384-ggOyR0iXCbMQv3Xipma34MD+dH/1fQ784/j6cY/iJTQUOhcWr7x9JvoRxT2MZw1T" crossorigin="anonymous">
        <link rel="stylesheet" href="{% static 'bootstrap.min.css' %}" />
        <script src="https://code.jquery.com/jquery-3.3.1.slim.min.js" integrity="sha384-q8i/X+965DzO0rT7abK41JStQIAqVgRVzpbzo5smXKp4YfRvH+8abtTE1Pi6jizo" crossorigin="anonymous"></script>
        <script src="https://cdnjs.cloudflare.com/ajax/libs/popper.js/1.14.7/umd/popper.min.js" integrity="sha384-UO2eT0CpHqdSJQ6hJty5KVphtPhzWj9WO1clHTMGa3JDZwrnQq4sF86dIHNDz0W1" crossorigin="anonymous"></script>
        <script src="https://stackpath.bootstrapcdn.com/bootstrap/4.3.1/js/bootstrap.min.js" integrity="sha384-JjSmVgyd0p3pXB1rRibZUAYoIIy6OrQ6VrjIEaFf/nJGzIxFDsf4x0xIM+B07jRM" crossorigin="anonymous"></script>
    </head>
    <body>
        <div class="container">
            {% block contents %}
            {% endblock %}
        </div>
        
    </body>
</html>


base.html을 상속받아 사용한 login.html 파일이다. extends 템플릿 태그를 사용하여 상속을 받은 후 {% block contents %} ~ {% endblock %} 사이에 동적으로 변하는 코드를 작성하였다.

login.html
{% raw %}
{% extends "base.html" %}

{% block contents %}
    <div class="row mt-5">
        <div class="col-12 text-center">
            <h1>로그인</h1>
        </div>
    </div>
    <div class="row mt-5">
        <div class="col-12">
            {{error}}
        </div>
    </div>
{% endblock %}


  • {% for field in form %} : view로부터 받은 form 값을 반복문을 이용해 field에 사용하고 있다
    form에는 username과 password의 field가 있으므로 반복문을 돌릴 경우 2가지가 나오게 된다.
  • {{ field.id_for_label }} : id_username 과 같은 값이다
  • {{ field.label }} : forms.py에서 지정해줄 수 있는 값이다
  • {{ field.field.widget.input_type }} : forms.py에서 지정해준 widget값이다
  • {% if field.errors %} : username 과 password에서 에러가 발생하면 설정한 color로 에러 문구를 출력해준다
  • {{ field.errors }} : 에러 정보 출력해준다
login.html
{% raw %}
 {% for field in form %}
                <div class="form-group">
                    <label for="{{ field.id_for_label }}">{{ field.label }}</label>
                    <input type="{{ field.field.widget.input_type }}" class="form-control" id='{{ field.id_for_label }}' placeholder="{{ field.label }}"name='{{ field.name }}'>
                </div>
                {% if field.errors %}
                <span style="color:tomato">{{ field.errors }}</span>
                {% endif %}

                {% endfor %}

field.errors 로 id나 pwd가 유효한 값인지 아닌지 출력해주는 문장은 MTV 생성에 View 부분을 참고하자 <click📌>

forms.py에서 설정한대로 화면에 구현이 된것을 볼 수 있다 !
