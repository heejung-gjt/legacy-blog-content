virtualenv 가상환경에서 네이버 영화 제목,코드 스크래핑하기

가상환경(Virtual Environment)

내 컴퓨터 내에 특정 공간에서만 사용하도록 만드는 제한된 공간을 만드는 것으로 컴퓨터 내의 독립된 환경을 만들어 로컬 전역에 설치하지 않아도 패키지를 사용할 수 있다. 사용하는 이유는 내 컴퓨터에 설치되어있는 버전에 방해받지 않고 개발 환경을 맞출 수 있다는 장점이 있다


virtualenv 설치

  • pip3 install virtualenv
  • mkdir testproject
  • cd testproject
  • virtualenv venv : venv(=컨벤션)의 가상환경 설정
  • source venv/bin/activate : venv 가상환경 실행
  • pip install requests : 원하는 패키지 설치
  • pip freeze>requirements.txt : 패키지들의 버전이 명시된 txt 파일 생성됨
  • pip install -r requirements.txt : 명세를 기준으로 패키지들을 설치

1. 네이버 뉴스 제목, 링크 스크래핑

-> find사용한 실습

import requests
from bs4 import BeautifulSoup

url = 'https://search.naver.com/search.naver?where=nexearch&sm=top_hty&fbm=1&ie=utf8&query=%EC%BD%94%EB%A1%9C%EB%82%98'
response = requests.get(url)

if response.status_code == 200:
html = response.text
soup = BeautifulSoup(html, 'html.parser')

ul = soup.find('ul',{'class':'list_news'}) # ul이 배열이 아니므로 ul안에 있는 li를 요소로 가지고 오는 배열을 find_all을 통해 가져온 후 lis에 넣어준다 그럼 lis를 for문을 돌려 print할 수 있다
lis = ul.find_all('li',{'class','bx'})

for li in lis:
print(li.find('a','news_tit').text)
print(li.find('a',{'class':'news_tit'}).text)
print(li.find('a',{'class':'news_tit'})['href'])

import requests
