👉 Data Types


[Task]

  1. i,d,s 변수를 각각 int, double, string으로 선언한다
  2. 3개의 변수를 입력형식(input format)에 따라 입력하고 초기화한다
  3. +연산자를 사용하여 다음 작업을 수행해라
    ① 첫번째 줄에는 i의 합계를 출력해라
    ② 두번째 줄에는 d의 합계를 소수점 첫째자리까지 출력해라
    ③ 세번째 줄에는 s의 문자열과 입력 문자열을 연결해라
  4. yearPasses 함수에는 age 변수가 1씩 증가하는 값을 작성한다

[Input Format]


[Constraints]

1 <= T <= 4
-5 <= age <= 30


[Output Format]


[Sample Input]

12
4.0
is the best place to learn and practice coding!


[Sample Output]

16
8.0
HackerRank is the best place to learn and practice coding!


[Explanation]

정수를 더하면 4+12 = 16
부동 소수점을 더하면 4.0 + 4.0 = 8.0
HackerRank와 연결하면 HackerRank is the best place to learn and practice coding!


[Code]

i = 4
d = 4.0
s = 'HackerRank '
# Declare second integer, double, and String variables.

user_i = int(input())
user_d = float(input())
user_s = input()

# Read and save an integer, double, and String to your variables.
i_sum = i+user_i
d_sum = d+user_d
s_concat = s+user_s

# Print the sum of both integer variables on a new line.
print(i_sum)

# Print the sum of the double variables on a new line.
print('%.1f' % d_sum)

# Concatenate and print the String variables on a new line
# The 's' variable above should be printed first.
print(s_concat)
