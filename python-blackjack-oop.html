BlackJack 게임 class 설계와 구현

👉 규칙

공통

  1. 카드를 받는다
  2. 카드를 뽑는다
  3. 뽑은 카드를 가져간다
  4. 카드를 오픈한다


유저

  1. 카드를 더 받을지 안받을지를 선택한다
  2. 카드를 한번더 섞을지 안섞을지 결정한다


딜러

  1. 2개의 카드합이 16이하이면 16을 넘을때까지 카드를 계속해서 뽑는다
  2. 카드를 나누어 준다


카드

  1. 40개의 카드덱을 만든다
  2. 랜덤으로 3번 섞는다
  3. 유저와 딜러가 카드를 뽑을때마다 카드덱의 개수는 줄어든다
