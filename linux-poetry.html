우분투에 poetry설치 후 장고 사용을 위한 가상환경 구축하기

👉 pyenv, virturalenv 설치하는 방법보기

✔ Ubuntu 20.04.1 LTS에서 poetry 설치하기

poetry.org site
위의 사이트를 들어가면 os마다 installation이 친절히 써져있다.일단 여기서는 linux를 기준으로 설치를 하는 방법이다.

1. 기본 터미널 들어가 curl로 시작하는 명령어를 install해준다.

curl -sSL https://raw.githubusercontent.com/python-poetry/poetry/master/get-poetry.py | python -


2. 설치가 끝나면 poetry가 잘 설치되었는지 확인한다.

poetry --version


이때 저자는 여기서 오류가 발생하였다.. 왜 넣는지 정확한 이유는 모르겠지만 아래와 같은 코드를 ./bashrc에 붙여넣는다.
(이때 vim ./bashrc는 root에서 실행시켜야 가능하다) 이후 적용을 위해 source ~/.bashrc로 새롭게 업데이트해준다

export PATH="$HOME/.poetry/bin:$PATH"


3. 잘 설치었다면 장고를 위한 가상환경을 구축해준다.

1. pyenv가 있는지 확인한다.

2. 가상환경 생성 후 폴더를 만들어준다 (순서는 상관x)

3. 패키지 관리 위해 poetry 실행시켜준다

📌 가상환경 만들기 - 실습

✔ 새로운 디렉토리를 만든다

$ mkdir module-practice


✔ 새로 만든 디렉토리로 들어가서 pyenv virtualenv (version) (env name)로 원하는 버전으로 가상환경을 만든다

$ pyenv virturalenv 3.9.1 py_env


✔ 가상환경을 실행시켜준다 pyenv activate (env name)

$ pyenv activate py_env


✔ 해당 디렉토리에서 poetry를 실행시켜준다. pyproject.toml파일이 생성된다.

(py_env) $ poetry init


✔ 그 후 poetry add로 원하는 패키지들을 설치해준다 .lock파일이 생성되고 해당 파일의 pyproject.toml파일과 같은 위치에 있어야 한다.

(py_env) $ poetry add dajngo==1.10
  • 패키지 제거하는 방법은 poetry remove <해당 프로그램>이다.
