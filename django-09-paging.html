게시판 paging처리를 하기 위해서는 작성된 게시글의 모든 object의 개수를 가져와야 하는데 이때 queryset이 사용된다
board_all = Board.objects.all().order_by('-id') 이 코드가 이해가지 않으면 먼저 공부하자


페이징이란

한 화면에 표시할 게시물이 많으면 스크롤바를 내려야 하는 불편함이 생김으로 이를 개선하기 위해 페이지를 원하는 개수로 나누어 추가하는 것이다.

  • 📌페이징 처리를 할 template에 출력될 위치를 정해 작성한다

  • 📌views.py에 django에서 제공하는 paginator을 사용한다

    paginator는 페이지당 몇개의 글을 화면에 보여줄것인지 도와주는 모듈이다

① 작성한 게시글 객체를 모두 가져와 board_all 변수에 저장한다

② import한 Paginator을 사용한다. Paginator(페이지로 분할될 객체 , 한 페이지에 담길 객체의 수)를 받고 있다 즉, Board에 있는 모든 객체를 한페이지에 2개씩 출력하겠다는 이야기이다

③ request는 사용자가 보낸 모든 정보를 담고 있으며 GET방식으로 정보를 받아온 데이터를 가리킨다. 해당 데이터를 딕셔너리의 get으로 (key값으로 value값 찾는 함수) page의 value값을 정수형으로 받아온다 즉 page의 첫번째 페이지에 있는 값들을 화면에 출력하게 된다

④ get_page 메소드가 page의 값을 받아 해당 페이지를 화면에 출력해준다

board/views.py
from django.core.paginator import Paginator

def board_list(request):
    board_all = Board.objects.all().order_by('-id') - ⓛ
    paginator = Paginator(board_all, 2)  - ②
    page = int(request.GET.get('page','1')) - ③
    boards = paginator.get_page(page) - ④
    return render(request, 'board_list.html',{'boards': boards})

필요한 정보들은 모두 board_list 템플릿으로 전달하였다 (페이지 값)


  • 📌template에 페이징한 글 출력하기

① {% if boards.has_previous %} : 이전 페이지가 존재하는지의 여부를 확인하기 위하여 조건문을 작성한다

② href="?p={{ boards.previous_page_number }} : 만약 이전 페이지가 존재하면 버튼을 눌렀을때 이전페이지로 url을 이동시킨다

③ {{ boards.number }} / {{ boards.paginator.num_pages }} : 현재 페이지수 / 전체 페이지수

board/board_list.html
{% raw %}

<div class="row mt-12">
    <div class="col-12">
        <nav>
            <ul class="pagination justify-content-center">
                {% if boards.has_previous %} - ①
                <li class="page-item">
                    <a class="page-link" href="?p={{ boards.previous_page_number }}">이전으로</a> - ②
                </li>
                {% else %}
                <li class="page-item disabled">
                    <a class="page-link" href="#">이전으로</a>  # 존재하지 않으면 href="#"처리를 해준다
                </li>
                {% endif %}
                <li class="page-item active">
                    <a href="#" class="page-link">{{ boards.number }} / {{ boards.paginator.num_pages }}</a> - ③
                </li>
                {% if boards.has_next %} - ④
                <li class="page-item">
                    <a class="page-link" href="?p={{ boards.next_page_number }}">다음으로</a> - ⑤
                </li>
                {% else %}
                <li class="page-item">
                    <a class="page-link disabled" href="#">다음으로</a>
                </li>
                {% endif %}
            </ul>
        </nav>
    </div>
</div>
