1-1 클래스

1.클래스란

클래스를 사용하는 이유는 복잡하고 반복되는 기능을 다루기 쉽도록 만들기 위해 사용한다
클래스의 이름을 지정할때는 첫 문자마다 대문자를 쓰는 CapWords명으로 작성한다
클래스 내의 함수에는 메서드가 작성이 된다. 이때 초기화가 필요한데 init (self, ….)를 이용하여 초기화 해준다 self는 항상 객체 자신을 의미하는 파라미터이고 초기화 할 때 항상 작성해주어야 한다

메서드 : 메서드는 클래스 내부의 함수라고 할 수 있다. 메서드는 첫번째 파라미터에 self라는 파라미터를 갖고 이를 인스턴스 변수 라고한다
메서드 앞에는 항상 self를 적어주어야 한다

인스턴스 변수 : 하나의 클래스로부터 여러 객체 인스턴스를 생성해서 사용할 수 있다 . 이 변수는 self. 을 사용하여 클래스 내부에서 사용이 가능하다.
클래스 밖에서 인스턴스 변수를 사용하기 위해서는 객체변수.인스턴스변수로 액세스 하여 사용할 수 있다

python의 클래스는 public가 기본값이기 때문에 특정 변수명이다 메서드를 private로 만들기 위해서는 두개의 밑줄(__)을 이름 앞에 붙이면 된다

class PersonInfor:   #python 네이밍 #문자마다 앞글자 대문자로 
    def __init__(self, name, sex):  #초기자 , self : 인스턴스 변수
        self.name = name
        self.sex = sex
        print('사람 정보 입력됨 \n')

    def personInfor(self, number):  #메서드
        print('성별이 {0} 인 {1} 씨의 번호는 {2} 입니다' .format(self.sex, self.name, number))

4-1 상속

1.상속

클래스를 상속 받기 위해서 자식 클래스에서 클래스명 뒤에 부모 클래스 이름을 괄호와 함께 넣어주면 된다.
아래의 코드에서는 Doll 클래스는 각각 부모 클래스인 Item으로부터 파생된 클래스이며 Item을 상속받고 있다

class Item:
    def __init__(self, name):
        self.name = name

    def price(self):
        print('부모 클래스의 price 가격 정보')
    
    def color(self):
        print('부모 클래스의 color 정보')

class Doll(Item):  #Item 상속받는다
    def color(self):
        print(' 자식 클래스의 color 정보')


doll = Doll('ZaDoo')   #크래스 Doll에는 name이 없지만 상속받고 있는 Item 클래스에는 name을 전달받으므로 name이 출력된다 
name = doll.name
print(name)
doll.color()  #Doll클래스에는 color함수가 있기 때문에 해당 함수가 실행된다
doll.price()  #Doll 클래스에는 price 함수가 없기 때문에 상속받고 있는 Item의 price 정보가 출력된다

'''
결과
ZaDoo
자식 클래스의 color 정보
부모 클래스의 price 가격 정보
'''


2.상속 예 - 게임 속 육지와 지상 탱크 아이템 생성하기

육지 탱크와 지상 탱크는 모두 name과 hp, damage를 갖는것은 동일하다. 이때 지상 탱크는 날 수 있는 기능과 진화된 공격력을 따로 가지므로 육지 탱크의 기능을 상속받아서 지상 탱크의 기능을 보충하면 된다

class Tank: #육지 탱크 클래스
    def __init__(self, name, hp, damage):
        self.name = name
        self.hp = hp
        self.damage = damage
    
    def create(self):
        print('육지 탱크{0} 생성 : 체력[{1}]  공격력[{2}]\n'.format(self.name, self.hp, self.damage))
    

class FlyTank(Tank):  #지상 탱크 클래스 
    def __init__(self,name, hp, damage, direction):
        Tank.__init__(self, name, hp, damage)  #Tank 클래스의 있는 인스턴스를 가져오기 위해 Tank.__init__...
        self.direction = direction

    def create(self, fly_damage):
        self.fly_damage = fly_damage
        print('지상 탱크{0} 생성 : 체력[{1}] 공격력[{2}] 지상 공격력[{3}]\n'.format(self.name, self.hp, self.damage, fly_damage))
    
    def attack(self):
        print('지상탱크 공격중....\n')
        print('지상 탱크{0} :{1} 방향으로 공격  공격력{2} 지상 공격력 [{3}]\n'.format(self.name, self.direction, self.damage, self.fly_damage))


tank01 = Tank('tank01', 50, 25)  #육지 탱크 생성
tank01.create()

flytank01 = FlyTank('flytank01', 100, 60, '6시')  #지상 탱크 생성 , 상속을 받았기 때문에 Tank에 있는 인스턴스도 함께 써줄 수 있다
flytank01.create(30)
flytank01.attack()

'''
결과 
육지 탱크tank01 생성 : 체력[50]  공격력[25]

지상 탱크flytank01 생성 : 체력[100] 공격력[60] 지상 공격력[30]

지상탱크 공격중....

지상 탱크flytank01 :6시 방향으로 공격  공격력60 지상 공격력 [30]
'''


3.다중 상속

다중상속이란 여러개의 클래스를 상속받는 것을 말한다

4.다중 상속 예 - 공격력이 없는 탱크와 날수 있는 탱크를 상속받는 공격력 있는 날수 있는 탱크 아이템 생성하기

CureTank는 name과 hp의 멤버변수를 가지고 있고 FlyTank는 fly_direction의 멤버변수를 가지고 있다 이때, AttackFlyTank 클래스가 이 두개의 클래스를 상속받는다

class CureTank:
    def __init__(self, name, hp):
        self.name = name
        self.hp = hp
    
    def create(self):
        print('육지 탱크{0} 생성 : 체력[{1}]\n'.format(self.name, self.hp))
    
class FlyTank:
    def __init__(self, fly_direction):
        self.fly_direction = fly_direction

class AttackFlyTank(CureTank,FlyTank):
    def __init__(self,name, hp, fly_direction, damage):
        CureTank.__init__(self, name, hp)  #CureTank의 멤버변수를 사용할 수 있게 정의한다
        FlyTank.__init__(self, fly_direction)  #FlyTank의 멤버변수를 사용할 수 있게 정의한다
        self.damage = damage

    def attack(self):
        print('지상 탱크{0} :{1} 방향으로 공격  공격력[{2}]\n'.format(self.name, self.fly_direction, self.damage))


flytank01 =AttackFlyTank('flytank01', 50, '6시', 40)
flytank01.attack()

# 결과 : 지상 탱크flytank01 :6시 방향으로 공격  공격력[40]
