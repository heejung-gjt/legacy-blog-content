(python) Scope의 global, local, nonlocal

Scope(유효범위)

스코프란 변수에 접근할 수 있는 범위이다.Local Scope에서 선언되는 변수는 바깥 Scope에서 사용 불가능하다. Global Scope는 어디서든 접근이 가능하다. 우선순위는 Local Scope > Global Scope로 같은변수가 각각에 존재할때 local 안에 있는 변수가 출력이 된다


전역변수

num_print함수에서 바깥에 전역변수 num이라는 변수를 지정을 하였다. num은 함수내에서도 사용가능하며 바깥에서도 가능하다. 즉 코드 전체에서 접근할 수 있다

num = 10
def num_print():
    print(num)  # 전역변수 10 출력

num_print()
print(num)  # 전역변수 10 출력
# 답 : 10
#      10


함수 안에서 전역변수 변경하기

num_print 함수안에서 num의 값을 20으로 변경하였지만 함수를 빠져나오면 num의 값이 10으로 출력되는 것을 볼 수 있다. num_print함수 안에 있는 변수 num은 지역변수이므로 전역변수인 num의 값을 변경 할 수 없다

num = 10
def num_print():
    num = 20
    print(num)  # 20 출력

num_print()
print(num)  # 전역변수 10 출력
# 답 : 20
#      10


global로 전역변수 사용하기

함수 num_print에서 global를 사용하면 전역변수 num =10 을 사용한다. 따라서 num = 20 으로 변경되는 것은 전역변수가 변경되는 것과 같으므로 20이 출력된다

num = 10
def num_print():
    global num = 20
    print(num)  # 20 출력

num_print()
print(num)  # 전역변수 10 출력
# 답 : 20
#      20


nolocal사용하기

nonlocal은 현재 함수의 지역변수가 지역 변수가 아니라는뜻이다. 즉 바깥쪽 함수의 지역변수를 사용한다. 따라서 함수B에서 함수A의 지역변수 num을 변경할 수있다

def A():
    num = 10
    def B():
        nonlocal num
        num = 100
    B()
    print(num)
A()
# 답 : 100
